Projeto 1

Detalhes

O Aplicativo de Gestão Financeira é uma plataforma simples e eficaz para que os usuários controlem suas finanças pessoais. Permite o registro de contas bancárias, transações e categorias financeiras, facilitando a organização e análise de gastos e receitas.

Tecnologias

O Aplicativo de Gestão Financeira é construído com base em tecnologias modernas, incluindo Node.js e NestJS para o desenvolvimento do backend, TypeScript para tipagem estática e qualidade de código, PostgreSQL para armazenamento seguro de dados financeiros, Prisma para simplificar a interação com o banco de dados, Docker para facilitar a implantação consistente e Swagger para documentação acessível da API. Além disso, o aplicativo segue rigorosamente padrões de arquitetura limpa e adere ao princípio de responsabilidade única do SOLID, garantindo um código de alta qualidade e fácil manutenção.

Projeto 2

Detalhes

O Aplicativo para Estúdio de Tatuagens é uma solução inovadora que simplifica o processo de agendamento de tatuagens, proporcionando uma experiência mais eficiente e conveniente para os clientes e tatuadores. Os clientes podem se cadastrar, fornecer suas informações e acessar um catálogo de artes disponíveis, juntamente com os horários disponíveis de cada tatuador no estúdio. Eles têm a liberdade de selecionar a arte desejada, escolher um horário conveniente e agendar uma sessão com o tatuador de sua escolha. Este aplicativo visa fortalecer o relacionamento entre tatuadores e clientes, tornando o processo de agendamento mais acessível e transparente.

Tecnologias

O aplicativo é desenvolvido com tecnologias modernas, incluindo Node.js em conjunto com o Express como framework, JavaScript como linguagem de programação, PostgreSQL e Prisma para armazenamento de dados e interação com o banco de dados, JWT para autenticação de usuários, e é suportado por uma bateria completa de testes unitários e end-to-end. Além disso, o aplicativo segue rigorosamente padrões de arquitetura limpa e o princípio de responsabilidade única do SOLID para garantir um código de alta qualidade e facilidade de manutenção.
